Informática Básica - Ensino Médio

Hardware

Hardware é toda a parte física do computador, ou seja, tudo aquilo que podemos tocar. Ele é composto por peças e dispositivos responsáveis por executar tarefas, armazenar dados, processar informações e permitir a interação entre o usuário e o sistema.

O hardware é dividido em quatro categorias principais, e entender cada uma delas ajuda a compreender como a tecnologia funciona no dia a dia.

1️⃣ Dispositivos de Entrada

São responsáveis por enviar informações para o computador.

  • Teclado: digitação de textos, comandos e atalhos.
  • Mouse: movimentação de ponteiro, seleção de arquivos, navegação.
  • Scanner: digitaliza documentos físicos.
  • Webcam: capta vídeos e imagens.

2️⃣ Dispositivos de Saída

Exibem informações para o usuário.

  • Monitor: exibe imagens, vídeos, programas e jogos.
  • Impressora: transforma documentos digitais em papel.
  • Caixas de som: reproduzem sons e músicas.

3️⃣ Dispositivos de Armazenamento

Guardam arquivos temporária ou permanentemente.

  • HD: grande capacidade, porém lento.
  • SSD: muito rápido, deixa o sistema mais ágil.
  • Pendrive: portátil, ideal para transportar arquivos.
  • Cartões SD: usados em celulares, câmeras e tablets.

4️⃣ Dispositivos de Processamento

São o “cérebro do computador”.

  • Processador (CPU): executa cálculos e comandos.
  • Placa-mãe: conecta todas as peças do sistema.
  • Memória RAM: armazena dados temporários enquanto programas estão abertos.
  • Placa de vídeo: cria imagens, animações e gráficos.

Exemplo prático:
Quando você abre um jogo: → A CPU calcula tudo → A RAM guarda dados temporários → A placa de vídeo cria a imagem → O monitor exibe

Quando você joga um jogo online:
  • A CPU processa os comandos;
  • A RAM mantém o jogo ativo;
  • A placa de vídeo cria os gráficos;
  • O monitor exibe as imagens;
  • O HD/SSD armazena o jogo.

Software

Software é tudo aquilo que não podemos tocar no computador: são programas, aplicativos, jogos e sistemas que dizem ao hardware o que fazer. Eles funcionam como “instruções digitais” que controlam cada ação do computador.

1️⃣ Software de Sistema

São responsáveis por controlar todo o funcionamento do computador.

  • Windows
  • Linux
  • macOS
  • Drivers de impressora / mouse / placa de vídeo

2️⃣ Software Aplicativo

São programas usados no dia a dia para resolver problemas.

  • Google Chrome — navegar na internet
  • Word — criar textos
  • Excel — fazer cálculos e tabelas
  • Photoshop — editar imagens
  • Jogos — entretenimento

3️⃣ Softwares Utilitários

Servem para manutenção do computador.

  • Antivírus
  • Desfragmentador
  • Compactadores (WinRAR, 7zip)

    • Exemplo prático: Quando você baixa uma foto no WhatsApp e abre no "Visualizador de Fotos", o WhatsApp é o aplicativo e o visualizador é outro software. Tudo isso só funciona porque o Windows (software de sistema) está rodando.

Windows Básico

O Windows é o sistema operacional mais utilizado no mundo. Ele é responsável por controlar todas as funções do computador e oferecer uma interface gráfica intuitiva para que o usuário consiga usar o sistema sem precisar digitar códigos.

Principais partes da interface:

  • Área de Trabalho: onde ficam atalhos, pastas e arquivos.
  • Barra de Tarefas: mostra programas abertos.
  • Menu Iniciar: central de programas, configurações e desligamento.
  • Explorador de Arquivos: usado para criar, mover e renomear pastas.

Funções importantes do Windows:

  • Instalação de programas e aplicativos
  • Configuração de internet, impressoras e dispositivos
  • Organização de arquivos
  • Criação de contas de usuários
  • Proteção com firewall e antivírus integrado

Atalhos essenciais:

  • Ctrl + C: copiar
  • Ctrl + V: colar
  • Windows + E: abrir arquivos
  • Windows + L: bloquear o computador
  • Alt + Tab: alternar entre janelas

Exemplo real:
Ao fazer um trabalho escolar, você pode baixar imagens pelo Chrome, organizar numa pasta e depois usar o Word. Tudo isso é gerenciado pelo Windows.

Internet

A Internet é uma rede mundial de computadores interligados que permite a comunicação e o compartilhamento de informações entre pessoas, empresas e sistemas.

Como a internet funciona?

Quando você acessa um site, seu computador envia uma solicitação para um servidor usando seu roteador. O servidor responde enviando textos, imagens e vídeos exibidos no navegador.

Principais serviços da internet:

  • E-mail: envio de mensagens formais.
  • Redes Sociais: Instagram, TikTok, Facebook.
  • Pesquisa: estudo usando Google.
  • Streaming: YouTube, Netflix, Spotify.
  • Compras Online: Mercado Livre, Amazon, Shopee.
  • Bancos Digitais: PIX, pagamentos, boletos.
  • Nuvem: Drive, OneDrive, Dropbox.

    • Exemplo real do dia a dia:
    Quando você posta uma foto no Instagram:

    • Faz upload da imagem para os servidores da Meta
    • O servidor salva a foto
    • Outras pessoas fazem download para visualizar

    Cuidados importantes na internet:

    • Não clicar em links suspeitos
    • Não fornecer senhas para desconhecidos
    • Evitar sites sem HTTPS
    • Usar senhas fortes
